============
Installation
============

Requirements
============

- TYPO3 v13 or v14
- PHP version compatible with your TYPO3 version

Install via Composer
====================

.. code-block:: bash

   composer require nitsan/ns-basetheme

Activation
==========

- Activate the extension in TYPO3 backend.
- Clear caches after installation.
